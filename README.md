# Customer Churn Prediction Streamlit App

This project is a simple machine learning web application built using Streamlit.  
It predicts whether a customer is likely to churn based on customer information.

## Features

- User input fields for customer details
- Churn prediction: Yes / No
- Prediction probability
- Simple explanation of model output
- Clean Streamlit user interface

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- NumPy
- Pickle
